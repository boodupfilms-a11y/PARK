# BOODUP PARK – GitHub Actions Cloud Build

**מטרה:** לתת לך APK מוכן בלי Ubuntu על המחשב שלך.

## שלבים קצרים
1) צור ריפו חדש ב-GitHub (פרטי או ציבורי).
2) העלה לשם את כל הקבצים מהתיקייה הזו.
3) עבור ל-Actions → הפעל ידנית את **Build Android APK** (Workflow Dispatch).
4) בסוף הריצה תראה למטה "Artifacts" בשם **BoodupPark-APK** – לחץ להורדה.
5) העבר את ה-APK לטלפון (Redmi 10 תומך arm64-v8a) והתקן.

> אם תרצה חתימה בשם חבילה קבוע (release keystore), נוסיף שלב בהמשך.
